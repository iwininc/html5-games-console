requestLogin

()

Defined in src/console-js/modules/social.js:21

This method should be called to explicitly request a login to facebook. The result function that is returned will be called with a value of either the user ID of who logged in, or null if the user isn't logged in. If the user is already logged in, they won't be presented with a login dialog. NOTE In the desktop console, this will request a login, but the callback will no later than 15 seconds after calling, as there is no way through our current integration to determine exactly if / when a user has signed in through this call.

Returns:

:

An object that contains a member function result that you should call with a callback function that will be fired when facebook has responded. The format of result function is

  • result( callbackFunction );
    Where callbackFunction is a function that will be called with a
  • single parameter, the user's facebook ID or null if not logged in

Example:

iConsole.social.facebook.requestLogin().result( function( userID ) {
    if ( userID ) {
        myGame.recordSignedIn( userID );
    } else {
        myGame.restartLevel();
    }
});

requestShare

(
  • data
)

Defined in src/console-js/modules/social.js:55

Request a share dialog is opened to share the current game/application. Games running in the normal console should not (currently) make this call, as the URL that will be shared will not allow the game to launch. However, games running in exclusive mode in a dedicated app should make this call.

Parameters:

  • data Object

    An object which must contain a key pair for 'message', the message you want to share with users who are playing

Returns:

:

An object that contains a member function result that you should call with a callback function that will be fired when facebook has responded. This callback function will be called with a single parameter, which will be an object with a success value of either true or false.

Example:

iConsole.social.facebook.requestShare( { message : 'Come play our awesome game' } ).result( function( data ) {
    if (data.success) {
        myGame.thankUserForSharing();
    } else {
        myGame.restartLevel();
    }
});
