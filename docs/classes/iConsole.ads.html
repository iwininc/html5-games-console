iConsole.ads Class

Defined in: src/console-js/modules/ads.js:2

Adverts

Adverts are controlled by the game_details.json file located in the root of game.

Types

  • 'none' No adverts
  • 'interstitial' Always show ads
  • 'levelStarted' dependant on 'level' numbers defined in the section
  • 'max_time' displays when 'maxseconds' has expired between calls.
  • 'every_attempt' displays at everyattempt from 'start' count then ever 'between' count

show

(
  • data
)
Object

Defined in src/console-js/modules/ads.js:27

In order to flag an advert to potentially be shown, you call this function with appropriate data blocks as defined below. You should pause your game play whilst the advert is displayed, and thus you should consider the advert to be displayed, until the console triggers the result member function of the return value.
If you declare your data type to be interstitial, then the advert (currently) will be triggered always. If you declare your data type to be levelStarted, then the advert may or may not fire based upon the behaviour defined in the game_details.json file. If the advert isn't fired, the result is fired immediately. In this case, any code appearing on lines after this call will be called after the result callback is fired, whereas if the advert is fired, any code appearing on lines after this call will be called before the result is fired. Therefore, you should consider pausing all game execution prior to this call, and leave this as the last call to make.

Parameters:

  • data Object

    An object which contains either

    {
   type: 'interstitial'
}
    to fire interstitial adverts all the time, or:

    {
   type: 'levelStarted',
   level: num
}
    where num is the level number as an integer that the user is about to start playing. In this instance the advert display behaviour is guided by the game_details.json file.

Returns:

Object:

An object that contains a member function result that you should call with a callback function that will be fired when the advert is dismissed. The format of result function is

result( callbackFunction );
Where callbackFunction is a function that will be called with a single parameter, an object that currently is empty, when the advert has dismissed.

Example:

iConsole.ads.show( {
   type: 'levelStarted',
   level: 3
} ).result( function( resultData ) {
   myGame.startGameTimer();
} );
